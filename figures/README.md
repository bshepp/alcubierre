# `figures/`

Standalone PNG renders of the project's headline data.

Generated by [`figures/plot_figures.py`](../figures/plot_figures.py)
(subcommand dispatch). Each subcommand reads a parquet from `sweeps/` /
`sweeps_remote/` or a CSV from `warp_factory_repro/` and writes one or more
PNGs into the corresponding `figures/<topic>/` subdirectory.

To regenerate everything:

```powershell
python figures/plot_figures.py all
```

Or one at a time:

```powershell
python figures/plot_figures.py fh-corner
python figures/plot_figures.py kappa-surface-3d
python figures/plot_figures.py thickness-heatmap
python figures/plot_figures.py krasnikov-collapse
python figures/plot_figures.py gw-recoil-cliff
python figures/plot_figures.py hybrid-wall-heatmap
python figures/plot_figures.py shift-families-bars
```

companion `_NOTES.md` files) must include its **slice scope** â€” the
parameters held fixed and the assumptions that bound the claim. Schematics
are explicitly labelled "schematic", not "derived result".
